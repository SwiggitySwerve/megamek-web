# Artillery Weapons

This folder contains artillery weapons organized by type:

- `artillery-pieces.ts` - Standard artillery pieces (Arrow IV, Long Tom, Sniper, Thumper)
- `artillery-cannons.ts` - Artillery cannons (Long Tom Cannon, Sniper Cannon, Thumper Cannon, Artillery Cannon)
- `mortars.ts` - Mortar systems (Mech Mortars, BA Tube Artillery)
- `index.ts` - Main export file that aggregates all artillery weapons
