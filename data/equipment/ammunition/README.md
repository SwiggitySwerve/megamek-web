# Ammunition

This folder contains ammunition organized by weapon type:

- `autocannon-ammo.ts` - Standard autocannon ammunition
- `ultra-autocannon-ammo.ts` - Ultra autocannon ammunition
- `lb-autocannon-ammo.ts` - LB-X autocannon ammunition
- `gauss-ammo.ts` - Gauss rifle ammunition
- `missile-ammo.ts` - Missile weapon ammunition (SRM, LRM, Streak)
- `machine-gun-ammo.ts` - Machine gun ammunition
- `index.ts` - Main export file that aggregates all ammunition
