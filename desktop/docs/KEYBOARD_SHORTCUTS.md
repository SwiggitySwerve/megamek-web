# BattleTech Editor - Keyboard Shortcuts

This document lists all keyboard shortcuts available in the BattleTech Editor desktop application.

## Platform Conventions

- **macOS**: Uses `Cmd` (âŒ˜) as the primary modifier
- **Windows/Linux**: Uses `Ctrl` as the primary modifier

## File Menu

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| New Unit | `Cmd+N` | `Ctrl+N` |
| Open | `Cmd+O` | `Ctrl+O` |
| Save | `Cmd+S` | `Ctrl+S` |
| Save As | `Cmd+Shift+S` | `Ctrl+Shift+S` |
| Import | `Cmd+I` | `Ctrl+I` |
| Export | `Cmd+E` | `Ctrl+E` |
| Print | `Cmd+P` | `Ctrl+P` |
| Preferences | `Cmd+,` | `Ctrl+,` |
| Quit | `Cmd+Q` | `Ctrl+Q` |

## Notes

1. **Undo/Redo**: Menu shortcuts take precedence over web app shortcuts when focused.
2. **Developer Tools**: Only available in development mode or when enabled in Advanced settings.
3. **Recent Files**: Up to 9 recent files can be accessed directly via keyboard shortcuts.
4. **Fullscreen**: Exit fullscreen using the same shortcut or `Esc` key.
