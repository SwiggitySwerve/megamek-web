# Capital Weapons

This folder contains capital weapons organized by type:

- `naval-lasers.ts` - Naval laser weapons (35, 45, 55)
- `naval-ppcs.ts` - Naval PPC weapons
- `naval-gauss.ts` - Naval gauss rifle weapons
- `n-gauss.ts` - N-Gauss rifle weapons
- `mass-drivers.ts` - Mass driver weapons
- `sub-capital.ts` - Sub-capital weapons (cannons and lasers)
- `capital-missiles.ts` - Capital missile weapons
- `index.ts` - Main export file that aggregates all capital weapons
