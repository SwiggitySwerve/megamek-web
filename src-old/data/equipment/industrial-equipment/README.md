# Industrial Equipment

This folder contains industrial equipment organized by type:

- `construction-equipment.ts` - Construction and excavation equipment (backhoe, bulldozer, chainsaw, etc.)
- `salvage-equipment.ts` - Salvage and repair equipment (salvage arm, lift hoist, spot welder)
- `support-equipment.ts` - Support and utility equipment (fuel tanks, ladders, bridge layers)
- `index.ts` - Main export file that aggregates all industrial equipment
