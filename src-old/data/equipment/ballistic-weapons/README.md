# Ballistic Weapons

This folder contains ballistic weapons organized by type:

- `gauss.ts` - Gauss rifles and variants
- `autocannons.ts` - Standard autocannons and variants  
- `lb-autocannons.ts` - LB-X autocannons
- `ultra-autocannons.ts` - Ultra autocannons
- `machine-guns.ts` - Machine guns and anti-missile systems
- `index.ts` - Main export file that aggregates all ballistic weapons
